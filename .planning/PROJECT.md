# Project: Rogue Like RPG

## Overview
This project aims to develop a rogue-like RPG game in pygame.

## Core Value
Provide an engaging and replayable dungeon-crawling experience with strategic turn-based combat.

## Milestones
- **Foundation & Combat (Phases 1-4):** Complete. Implemented basic ECS, rendering, turn-based movement, and combat.
- **Nested Worlds & Navigation (Phases 5-6):** Complete. Implemented nested world architecture, portals, and advanced navigation UI.
- **Layered Rendering & Structure (Phase 7):** Complete. Improved visual presentation with selective layer rendering and structural map elements.
- **Procedural Map Features (Phase 8):** Complete. Transitioned map creation from static coordinates to modular generators for buildings and environment details.
- **Data-Driven Architecture & Investigation Prep (Current):** Refactor to data-driven design for tiles, maps, and entities; prepare description system.
